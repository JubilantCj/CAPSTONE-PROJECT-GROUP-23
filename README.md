Frontend Capstone Project – Group 23
Project Overview
This project is a React-based web application focused on exploring our solar system. The application recreates a provided mockup, fetches real planetary data from an external API, displays it in a responsive grid of planet cards, includes a comparative NASA data table, and features a validated contact form.
The project demonstrates the use of HTML, CSS, JavaScript, React, API integration, responsive design, and collaborative GitHub workflows.
Of the 18 students assigned to Group 23, 11 joined the project's collaboration group and were added to the GitHub repository. Ten members actively contributed to the development of the project, while one member remained a collaborator but did not participate in development.

Code Review and Collaboration
Development was coordinated through GitHub and WhatsApp. Team members worked individually or in pairs on assigned components, submitted pull requests, and participated in code reviews before merging changes into the main branch.
The Team Lead, Efeadi Micheal [https://github.com/JubilantCj], and the Assistant Team Lead, Moremi Adekunle [https://github.com/DevMoremi], reviewed pull requests and ensured code quality, consistency, and adherence to project requirements before approvingl merges.
